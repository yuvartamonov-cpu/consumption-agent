# Lottery Agent

Журнал участия в лотереях Столото. Бюджет: **1000 ₽/неделя**.

## Стратегия

Принцип: диверсификация по бинго-лотереям, никаких быстрых игр (Рапидо/Кено).

| День | Игра | Билеты | Стоимость |
|---|---|---|---|
| Вс | Русское лото | 2 × 150 ₽ | 300 ₽ |
| Вт | Жилищная лотерея | 2 × 150 ₽ | 300 ₽ |
| Чт | Золотой бочонок | 2 × 150 ₽ | 300 ₽ |
| Любой | Спортлото 6/45+ | 1 × 100 ₽ | 100 ₽ |
| | **Итого** | | **1000 ₽** |

_Спортлото добавлено 26.04.2026 после выигрыша 500 ₽ на тираже №324 (билет №61)._

## Файлы

- `tickets.csv` — журнал билетов и результатов
- `weekly-plan.md` — текущий план недели
- `reports/` — месячные отчёты

## CSV-схема

```
date,game,draw,ticket_number,cost,status,prize,notes
```

- `date` — дата покупки (YYYY-MM-DD)
- `game` — `ruslotto` | `gzhl` | `bochonok` | `spec` | др.
- `draw` — номер тиража
- `ticket_number` — номер билета (с сайта/чека)
- `cost` — стоимость в ₽
- `status` — `pending` | `lost` | `won`
- `prize` — выигрыш в ₽ (0 если проигрыш)
- `notes` — свободный текст

## Стоп-сигналы

- Месячный проигрыш > 3000 ₽ → пауза, переоценка стратегии.
- Любой выигрыш > 4000 ₽ → напомнить про НДФЛ 13%.

## Что делает агент

1. Воскресенье утром — напоминание с планом покупок на неделю.
2. После каждого тиража — запрос номеров купленных билетов, проверка через сайт, запись в `tickets.csv`.
3. Конец месяца — отчёт в `reports/YYYY-MM.md` (вложено / вернулось / ROI).
4. Контроль стоп-сигналов.

## Как пользоваться

Юрий покупает билеты вручную на stoloto.ru, скидывает агенту:
- номер билета
- номер тиража
- игру

Агент записывает и проверяет результаты.
