
Как найти ссылки на чеки в Gmail:

1. Открой Gmail в браузере
2. Введи в поиск: 
   from:sender.ozon.ru subject:"Ваш чек"
3. Открой любое письмо
4. В теле письма найди ссылку вида:
   https://www.ozon.ru/my/e-check/download/XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX
5. Скопируй все такие ссылки в файл links.txt в этой папке

Я покажу как их скачать через браузер.
