### Исправление кнопки "Удалить" (Memory Lane)
- Баг: FOREIGN KEY constraint failed при DELETE FROM memory_lane_items
- Причина: `ml_reminders` и `ml_watchlist` ссылаются на `memory_lane_items(id)` через FK
- В БД нет ON DELETE CASCADE, поэтому удаление падает
- **Исправление в callbacks.py**: перед удалением из memory_lane_items:
  1. `DELETE FROM ml_reminders WHERE item_id = ?`
  2. `DELETE FROM ml_watchlist WHERE item_id = ?`
  3. `PRAGMA foreign_keys=OFF` на время удаления (запасной вариант)
  4. Удаление из memory_lane_items
  5. Удаление файла медиа, если есть
  6. Подтверждение: "🗑 Запись #N удалена"
