## Memory Lane search

- Зафиксировано исходное состояние поиска товаров из Memory Lane как baseline: старый `consumption_agent/ml_search.py` был слишком примитивен.
- За сессию переработан рабочий `ml_search_v2` / `ml_providers.py` в сторону seller-link search вместо зависимости от Ozon.
- Новая логика:
  - при распознанном бренде используется строгий brand-gated поиск;
  - бренд в provider-запросах берётся в кавычки;
  - выдача с явным чужим брендом отбрасывается;
  - direct links строятся на Lamoda, Brandshop, Sneakerhead, DNS, Citilink, М.Видео, Hoff, Mr.Doors, IKEA, Goldapple, Иль де Ботэ, Яндекс.Маркет, Wildberries;
  - для `AliExpress` и `Alibaba` запросы переводятся с русского на английский;
  - для брендов добавляется fallback на поиск официального сайта через Google/Yandex;
  - link-only результаты больше не схлопываются в одну карточку.
- Практическая позиция Юрия: Ozon не считать опорным источником, основной упор делать на прямые ссылки продавцов и официальные/дистрибьюторские сайты.
