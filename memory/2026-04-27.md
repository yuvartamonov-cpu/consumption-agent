## Ozon — заказы не извлекаются из HTML (2-я попытка)

- Юрий дважды сохранял страницу заказов Ozon как HTML («Веб-страница, полностью») и присылал файлы.
- Оба раза HTML содержал только CSS-переменные и стили — **ни данных заказов, ни __NUXT__.state**.
- Подтверждено: Ozon — SPA на Nuxt.js; данные заказов подгружаются через GraphQL/fetch API после рендера, не попадают ни в исходный HTML, ни в __NUXT__.state, ни в localStorage.
- Юрий устал от попыток. Нужен простой путь: скриншот (PrtScn) страницы заказов → я вручную внесу данные.
- Вывод: автоматизация получения заказов Ozon откладывается; на MVP достаточно ручного ввода.
