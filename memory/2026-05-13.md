**`/fines`** (существующая, расширена):
- Теперь принудительно запускает `fines_bot.py --days 7 --check-sms` перед показом данных
- Показывает неоплаченные штрафы с кнопкой ✅ Оплачено

**`/debts`** (существующая, расширена):

  - 🟡 СРОЧНО (days ≤ 3)
  - 🟢 На этой неделе (days ≤ 7)
  - ⚪ Остальные
- Итоговая сумма всех платежей
- Предупреждение о срочных платежах

- Выводит:
  - Сводку: количество покупок и общая сумма
  - Построчный список: магазин → сумма → описание
  - Итог по магазинам
- Markdown-форматирование (esc_md)
- Зарегистрирована в `add_handlers`
- Добавлена в `/help`

**`/help`** — добавлены строки:
