# 2026-05-21

- `consumption_agent` / Memory Lane search shifted away from literal translation: `ml_translate.py`, `ml_providers.py`, and `ml_search_v2.py` now build shorter semantic/visual product queries from image-derived attributes for foreign search instead of noisy word-for-word text. This was prompted by bad results like the `Buckingham Palace` case.
- `Memory Lane` search UI now shows top results as separate URL buttons in `/ml_search` instead of only inline text links; pagination and watchlist controls remain below them.
- Receipt ingestion now runs inline LLM category classification for new receipt items in `receipt_pipeline.py` with safe fallback rules: confident matches map to existing categories, delivery stays `service`, weak/odd suggestions do not auto-create categories.
- Low-confidence receipt categorization now enters an interactive Telegram review flow: the bot offers 3 category options, the user can choose or reject them, and after 3 rejects the item is kept in `Прочее`.
