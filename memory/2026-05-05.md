## Consumption Agent — вечерние доработки

### Бренд: всегда показывать
- `parse_clothing_tag` теперь всегда пишет `Бренд: ... / не найден`
- Если бренда нет — предупреждение: нужна часть бирки с логотипом

### Размер из рамки бирки
- Добавлена функция `_extract_tag_size_from_image(image_path)`
- Crop правой зоны (x 0.63-0.81, y 0.42-0.58) с масштабированием x6
- Tesseract с psm 6 и 11, whitelist цифр
- Проверка: 42 на ETRO, 40 на Massimo Dutti — оба находит

### Massimo Dutti OCR (плохо читаемый логотип)
- Добавлен crop-OCR по наклейке: 3 зоны (вся наклейка, артикул, цена), тексты аппендятся к ocr_text перед парсингом
- Квази-fuzzy для бренда: `MASS|MOSSI|MAR... DUTT` → `MASSIMO DUTTI`
- Артикулы формата `\d{4}/\d{3}/\d{3}` (0154/277/400) — поддержка

### Цена (терялась первая цифра)
- Собираются все price-кандидаты, выбирается максимальная реалистичная (> 1.0)
- Исправлено: 4.90 → 64.90 GBP на Massimo Dutti

### Вывод чеков структурированный (как бирки)
- `🧾 Чек распознан`
- Дата, Сумма, Товары (список с ценами/количеством)
- Если товары не найдены — `Товары: не найдены` + команда /add

### Поиск картинок
