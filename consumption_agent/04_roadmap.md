*Проблема:* Один и тот же заказ может быть загружен двумя способами:

*Гипотеза:* По совпадению суммы + даты + источника можно определить дубли.

   - ⬜ Обобщённый механизм для всех источников (Ozon, Самокат, Лавка, Еда, Драйв)
   - ⬜ UI: пометка "дубль" в /list и отчётах
   - ⬜ Автоматическое слияние: items из скриншота + чек = полная запись
