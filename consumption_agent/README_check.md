# Проверка consumption_agent (04.05.2026)

## Файлы для проверки
- `consumption_agent_040526.tar.gz` (архив с `telegram_bot.py` и `consumption_agent_full_030526.py`).

## Задачи
1. **OCR + QR-коды** (`telegram_bot.py`):
   - Проверь корректность парсинга чеков (Tesseract + pyzbar).
   - Как обрабатываются ошибки (например, отсутствие Tesseract)?

2. **Импорт чеков из Ozon** (`consumption_agent_full_030526.py`):
   - Работает ли фильтр `--sender ozon`?
   - Как обновляются куки для авторизации?

3. **Безопасность**:
   - Нет ли SQL-инъекций (используются ли параметризованные запросы)?
   - Как обрабатываются пользовательские данные (например, названия товаров)?

4. **Логика категорий**:
   - Корректно ли определяется категория (например, "корм" → "Животные")?
   - Можно ли расширить список ключевых слов?

5. **Обработка ошибок**:
   - Что произойдёт, если:
     - Нет интернета при импорте чеков?
     - БД заблокирована другим процессом?
     - Фото чека не содержит текст/QR-код?

## Как проверить
1. Распакуй архив:
   ```bash
   tar -xzf consumption_agent_040526.tar.gz
   ```
2. Запусти тесты:
   ```bash
   python3 -m pytest tests/  # Если есть тесты
   ```
3. Проверь вручную:
   - Запусти Telegram-бот и отправь фото чека.
   - Запусти импорт чеков из Ozon (`--sender ozon`).

---
**Приоритет**: Проверка OCR и безопасности.
