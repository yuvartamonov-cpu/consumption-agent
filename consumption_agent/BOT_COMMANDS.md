## 📋 Основные команды

### /start
Запуск бота. Показывает приветственное сообщение и основное меню.

### /help
Показать список всех доступных команд с кратким описанием.

---

## 🛒 Инвентарь и покупки

### /list
Показать инвентарь товаров по категориям.

**Вывод:**
- Список категорий с количеством товаров
- Общее число активных товаров

### /add `<name>` `[price]` `[category]`
Добавить товар в инвентарь вручную.

**Примеры:**
```
/add Кофемашина 15000 Техника
/add Наушники
```

### /items `[N]`
Показать последние N товаров из инвентаря (по умолчанию 10).

### /items_full `[N]`
Показать последние N товаров с полной информацией:
- Название, бренд, категория
- Цена, дата покупки
- Гарантия, срок замены
- Фото (если есть)

### /add_item
Интерактивное добавление товара пошагово:
1. Название
2. Бренд (опционально)
3. Категория
4. Цена
5. Дата покупки
6. Срок замены (месяцев)

## 🧾 Чеки и расходы

### /dayexp `[дата]`
Расходы за указанный день (по умолчанию — сегодня).

**Функции:**
- Сканирование всех 4 почт (Gmail, Yandex, Mail.ru ×2)
- Сканирование SMS с двух телефонов
- Группировка по магазинам
- Итоговая сумма

**Пример:** `/dayexp 2026-05-15`

### /monthexp `[месяц]`
Расходы с 1 числа текущего месяца по указанную дату.

**Функции:**
- Сканирование почт и SMS
- Группировка по дням и магазинам
- Общие траты с начала месяца

**Пример:** `/monthexp 2026-05-15`

### /parse `<текст чека>`
Распознать чек из текста (OCR).

**Пример:**
```
/parse Магнит
Молоко 89.50
Хлеб 45.00
ИТОГО: 134.50
```

## 🧠 Memory Lane

### /ml_last `[N]`
Показать последние N записей Memory Lane (по умолчанию 10).

**Вывод:**
- Текстовый список с ID, датой, темой, названием
- Фото с подписью (ID, название, описание, тема, дата)
- Кнопки: 🔍 Искать, ⏰ Напомнить, 🗑 Удалить

**Формат подписи к фото:**
```
🧠 ML #1
📌 Название товара
Описание...
📂 категория
🕒 2026-05-16
```

### /ml_search `<id>`
Поиск товара из Memory Lane на маркетплейсах.

**Пример:** `/ml_search 1`

**Результат:**
- Прямые ссылки на поиск в Ozon, Яндекс.Маркет, Wildberries
- (В разработке: умный поиск через рабочую модель)

### /ml_stats
Статистика Memory Lane:
- Общее количество записей
- Распределение по темам
- Последние добавления

### /topic_set `<слово>` `<тема>`
Добавить ассоциацию слово→тема для автоматической категоризации.

**Пример:** `/topic_set пальто одежда`

### /topic_list `[тема]`
Показать все правила ассоциаций (или по конкретной теме).

## 💳 Кредиты и штрафы

### /debts
Проверка кредитов и займов к оплате.

**Источники:**
- Email: Gmail, Yandex, Mail.ru ×2
- SMS: два телефона через Phone Link

**Вывод:**
- 🔴 ПРОСРОЧЕН
- 🟡 СРОЧНО (≤3 дня)
- 🟢 На этой неделе (≤7 дней)
- ⚪ Остальные

**Отслеживаемые кредиторы (26):**

Банки (16): Сбербанк, ВТБ, Тинькофф, Альфа-Банк, Совкомбанк, Райффайзен, Газпромбанк, Открытие, Росбанк, Уралсиб, Хоум Кредит, Ренессанс Кредит, Почта Банк, Ак Барс, Абсолют Банк, МДМ Банк

МФО (10): Turbozaim, Joy Finance, Nebus, Boostra, Эквазайм, Webzaim, Деньги Сразу, ФлэшЗайм, Фаст Финанс, Глацинт

### /fines
Проверка неоплаченных штрафов.

**Источники:**
- Почта Mail.ru (письма от Госуслуг)
- SMS

**Вывод:**
- Список штрафов с суммой и датой
- Кнопка ✅ Оплачено для подтверждения

## 📸 Работа с фото

### Добавление в Memory Lane
Отправь фото с подписью вида:
```
нравится #пальто #серое #зима
```

Бот автоматически:
1. Сохранит фото
2. Определит тему (одежда, мебель, техника...)
3. Распознает товар через Vision API (название, бренд)
4. Добавит в Memory Lane

### Добавление чека по фото
Отправь фото чека — бот распознает:
- Магазин
- Товары и цены
- Итоговую сумму
- Дату

### Распознавание предмета
Отправь фото предмета — бот предложит добавить в инвентарь.

## 🔧 Техническая информация

**База данных:** `consumption.db` (SQLite)  
**Локация:** `~/.openclaw/workspace/consumption_agent/`  
**Сервис:** `consumption-bot.service`  
**Логи:** `journalctl --user -u consumption-bot.service`

**Команды управления:**
```bash
systemctl --user start consumption-bot.service   # запуск
systemctl --user stop consumption-bot.service    # остановка
systemctl --user restart consumption-bot.service # перезапуск
systemctl --user status consumption-bot.service  # статус
```

## 📝 Примеры использования

**Сценарий 1: Добавить покупку**
1. Отправить фото чека → бот распознает
2. Или: `/add Кофе 450 Продукты`

**Сценарий 2: Проверить расходы**
1. `/dayexp` — сегодня
2. `/monthexp` — за месяц

**Сценарий 3: Сохранить понравившуюся вещь**
1. Отправить фото с подписью `нравится #платье #красное`
2. `/ml_last` — увидеть запись
3. Нажать 🔍 Искать — найти похожее

**Сценарий 4: Контроль кредитов**
1. Ждать автоматического уведомления (за 3 дня)
2. Или: `/debts` — принудительная проверка
3. Нажать ✅ Оплатил после оплаты
