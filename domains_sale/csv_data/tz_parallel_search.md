# ТЗ: Сбор контактов для холодной рассылки доменов

## Контекст
Продажа 4 премиум-доменов медицинской тематики от ООО «Контанго»:
- medtrade.ru
- gormed.ru
- medzakaz.ru
- medtender.ru

Нужно ~50 email-контактов потенциальных покупателей.

## Категории для поиска (в порядке приоритета)

### 1. Частные клиники и медцентры (~20)
Крупные сети Москвы и регионов: GMS Clinic, Европейский МЦ, Анадолу, Медскан, Будь Здоров, К+31, Come Back, Ниармедик, ОТТА, Семейный доктор, Детский доктор, клиники МГУ, Столица, Медицина Альфа, ЛДЦ, Доктор СМ-Клиника, Реавиз, Бальт, Добрый доктор, Мединцентр.

Искать email на сайтах в разделах "Контакты" / "Сотрудничество" / "Для партнёров". Если нет прямого email — искать форму обратной связи и пометить.

### 2. Поставщики медоборудования (~10)
Дистрибьюторы медтехники: Матрикс, Авиценна-Медика, Кодивек, МедСнаб, АктивМед, Тандер, Медкомплект, Медикус, Юнимед, Галилео-Медикс.

Искать email для отдела продаж или закупок.

### 3. Тендерные площадки (~10)
Площадки и организаторы госзакупок медоборудования, которые могут купить домен для тематического проекта: medzakaz.ru прямой конкурент таким площадкам.

### 4. Доменные инвесторы (~10)
Участники Telderi, nic.ru аукционов по меддоменам, владельцы портфелей premium-доменов .ru.

Искать через:
- объявления на Telderi (раздел "Куплю домен")
- профили на Domainers.ru
- известные ники домейнеров на nic.ru

## Формат результата
Один CSV-файл с колонками:
```
name,email,website,source,category,status,notes
```

**Пример строки:**
```
GMS Clinic,partner@gmsclinic.ru,https://gmsclinic.ru,site_parse,клиники,target,
К+31,,https://k31.ru,site_parse,клиники,no_email,только форма связи
ivanov_domainer,ivanov@gmail.com,https://telderi.ru/user/ivanov,auction,инвесторы,target,
```

## Правила
1. Только реальные email — без мусора (example@, @example.ru, service@, иконки)
2. Если email не найден — записать в статус `no_email`, указав причину
3. Не больше 5 email на одну компанию (лучше 1-2 основных)
4. Каждый email проверить — хотя бы что формат валидный

## Ожидаемый результат
CSV файл с ~50 контактами, готовый к импорту в скрипт рассылки.
